Лабораторную работу выполнил Балтинов М. А. Ит-13,14-24

Основные классы:<br>Main - Главный класс программы, содержащий точку входа и меню для выбора заданий <br>Test - Класс для тестирования функциональности, содержит методы для выполнения восьми различных заданий <br>Validator - Вспомогательный класс для валидации пользовательского ввода различных типов данных

Сущности: <br>Cat - Класс, представляющий кота с именем и возможностью мяукать <br>CounterMeow - Декоратор для подсчёта мяуканий, реализующий Meowable <br>Funs - Класс с методами для вызова мяуканий у объектов Meowable <br>Meowable - Интерфейс для объектов, способных мяукать <br>Participant - Класс, представляющий участника соревнований с фамилией, именем и баллами <br>methods - Класс с методами чтения участников из файла, заполнения карты баллов и вывода победителей <br>Cache - Обобщённый класс для кэширования значений с инвалидацией <br>Fraction - Класс, представляющий дробь с числителем, знаменателем и кэшированием вещественного значения <br>FractionOperations - Интерфейс для операций с дробями (получение значения, установка числителя/знаменателя) <br>Line - Обобщённый класс, представляющий линию с точками начала и конца <br>Point - Абстрактный класс базовой точки <br>PointX - Класс точки с координатой X <br>PointXY - Класс точки с координатами X и Y <br>Polyline - Класс ломаной линии с массивом точек и расчётом длины <br>PolylineBuilder - Класс для построения ломаной из коллекции точек с сортировкой и обработкой <br>PeopleGroup - Класс для обработки файла с именами и номерами, группировки по номерам <br>ListMerger - Обобщённый класс для слияния двух упорядоченных списков <br>MirroredQueue - Класс для создания очереди с зеркальным отражением списка <br>TextAnalyzer - Класс для анализа текста и подсчёта уникальных букв

Задание 1: Кэширование дроби<br>
Реализован класс Fraction с кэшированием вещественного значения через Cache. Демонстрируется создание дроби, получение значения, смена числителя/знаменателя с инвалидацией кэша.<br>

Задание 2: Количество мяуканий<br>
Реализован декоратор CounterMeow для подсчёта мяуканий кота. Демонстрируется мяуканье через Funs с выводом счётчика.<br>

Задание 3: Слияние списков<br>
Реализован обобщённый класс ListMerger для слияния двух упорядоченных списков. Демонстрируется на числах и строках с проверкой упорядоченности.<br>

Задание 4: Результаты соревнований<br>
Реализованы методы для чтения участников из файла, группировки по баллам и вывода топ-3 с учётом ничьих.<br>

Задание 5: Уникальные буквы в тексте<br>
Реализован класс TextAnalyzer для подсчёта уникальных букв в файле (игнорируя регистр).<br>

Задание 6: Зеркальная очередь<br>
Реализован класс MirroredQueue для создания очереди из списка с зеркальным отражением. Демонстрируется ввод списка и вывод очереди.<br>

Задание 7: Построение ломаной<br>
Реализован стрим в PolylineBuilder для обработки коллекции точек (убрать дубликаты, сделать Y положительными, отсортировать по X) и создания Polyline с расчётом длины.<br>

Задание 8: Группировка имён по номерам<br>
Реализован стрим в PeopleGroup для чтения файла, форматирования имён, фильтрации без номеров и группировки по номерам.<br>
