Отчет по лабораторной работе

Лабораторную работу выполнил Балтинов М. А. Ит-13,14-24

Основные классы:

Main - Главный класс программы, содержащий точку входа и меню для выбора заданий

Test - Класс для тестирования функциональности, содержит методы для выполнения восьми различных заданий

Validator - Вспомогательный класс для валидации пользовательского ввода различных типов данных

Сущности:

Кот и мяуканье (ru.baltinov.cats):

Cat - Класс, представляющий кота:

<ul>
<li>Имеет имя (String)</li>
<li>Конструктор проверяет, что имя не null и не пустое</li>
<li>Метод meow() выводит на экран "имя: мяу!"</li>
<li>Перегруженный метод meow(int times) выводит строку вида "имя: мяу-мяу-мяу!"</li>
<li>Переопределены toString, геттер getName</li>
</ul>

Meowable - Интерфейс, определяющий метод мяуканья void meow()

CounterMeow - Класс-декоратор (паттерн Декоратор), оборачивающий объект Meowable:

<ul>
<li>Считает количество вызовов meow()</li>
<li>Имеет методы getMeowCount() и resetMeowCount()</li>
<li>Переопределяет toString, делегируя decorated</li>
</ul>


Funs - Класс с утилитарными методами для мяуканья:

<ul>
<li>meowsCare(Meowable, int) вызывает meow() заданное количество раз</li>
<li>meowsCare(Meowable) вызывает meow() 5 раз</li>
</ul>

Дробь с кэшированием (ru.baltinov.fraction):

Fraction - Класс, представляющий дробь:

<ul>
<li>Имеет числитель (int) и знаменатель (int)</li>
<li>Конструктор проверяет, что знаменатель не 0, и обрабатывает отрицательные значения</li>
<li>Реализует интерфейс FractionOperations</li>
<li>Использует класс Cache для кэширования вещественного значения</li>
<li>При изменении числителя/знаменателя кэш инвалидируется</li>
<li>Переопределены toString, equals, hashCode</li>
</ul>


FractionOperations - Интерфейс с методами getDecimalValue() и setNumeratorAndDenominator(...)


Cache - Класс, реализующий шаблон Кэширования:

<ul>
<li>Принимает Supplier для вычисления значения</li>
<li>Сохраняет результат и валидность</li>
<li>getValue() возвращает кэшированное значение или вычисляет новое</li>
<li>invalidate() очищает кэш</li>
</ul>

Геометрия (ru.baltinov.geometry):

Point - Абстрактный базовый класс точки


PointX - Класс точки с координатой X:

<ul>
<li>Имеет свойство x</li>
<li>Переопределены toString, copy, equals, hashCode</li>
</ul>


PointXY - Класс точки на плоскости (наследует PointX):

<ul>
<li>Имеет свойства x (унаследовано) и y</li>
<li>Переопределены toString, copy, equals, hashCode</li>
</ul>


Line - Обобщенный класс линии:

<ul>
<li>Параметр типа T ограничен PointXY</li>
<li>Имеет начальную (PStart) и конечную (PEnd) точки</li>
<li>Конструкторы и сеттеры проверяют, что точки не null и не равны</li>
<li>Метод length() вычисляет длину линии</li>
<li>Переопределены toString, equals (учитывает прямой и обратный порядок точек), clone</li>
<li>Внутренний метод createCopy для защиты инкапсуляции</li>
</ul>


Polyline - Класс ломаной линии:

<ul>
<li>Содержит массив точек PointXY</li>
<li>Конструктор проверяет, что массив не null и содержит >= 2 точек</li>
<li>Метод calculatePolylineLength() вычисляет общую длину</li>
<li>Метод replacePoint() позволяет заменить точку по индексу</li>
<li>Переопределен toString</li>
</ul>


PolylineBuilder - Класс с фабричным методом:

<ul>
<li>buildPolyline(Collection<PointXY>) принимает коллекцию точек</li>
<li>Обрабатывает Y-координаты (делает положительными), убирает дубликаты, сортирует по X</li>
<li>Возвращает новый объект Polyline</li>
</ul>

Обработка конкурса (ru.baltinov.competiton):

Participant - Класс участника:

<ul>
<li>Имеет фамилию (String), имя (String), массив баллов (int[4])</li>
<li>Метод getTotalScore() вычисляет сумму баллов</li>
<li>Переопределен toString</li>
</ul>


methods - Класс с утилитарными методами:

<ul>
<li>readParticipantsFromFile(String) читает участников из файла</li>
<li>fillScoreMap(List) группирует участников по сумме баллов в TreeMap (в обратном порядке)</li>
<li>printWinners(TreeMap) выводит топ-3 (и связанных по баллам) участников</li>
</ul>

Слияние списков (ru.baltinov.lists):

ListMerger - Обобщенный класс для слияния упорядоченных списков:

<ul>
<li>Параметр типа T ограничен Comparable<T></li>
<li>Хранит два списка (firstList, secondList)</li>
<li>Метод mergeLists() объединяет два упорядоченных списка в один, проверяя упорядоченность</li>
<li>Метод isSorted() проверяет, упорядочен ли список</li>
<li>Переопределены toString, equals, hashCode</li>
</ul>

Обработка текста (ru.baltinov.set):

TextAnalyzer - Класс для анализа текста:

<ul>
<li>Содержит текст (String)</li>
<li>Метод countUniqueLetters() возвращает количество уникальных букв (без учета регистра)</li>
</ul>

Очередь-отражение (ru.baltinov.queue):

MirroredQueue - Класс очереди с "отражением":

<ul>
<li>Принимает список и создает очередь, содержащую элементы списка, а затем те же элементы в обратном порядке</li>
<li>Метод getQueue() возвращает копию очереди</li>
<li>Переопределен toString</li>
</ul>

Обработка файлов с людьми (ru.baltinov.group):

PeopleGroup - Класс с утилитарным методом:

<ul>
<li>processFile(String) читает файл, фильтрует строки с номерами, форматирует имена (первая заглавная), группирует по номеру</li>
</ul>

Задание 1: Дробь с кэшированием

Реализован класс Fraction, реализующий интерфейс FractionOperations. Добавлено кэширование вещественного значения с помощью вспомогательного класса Cache. При изменении числителя/знаменателя кэш инвалидируется. Демонстрируется разница между первым вычислением и последующим использованием кэша.

Задание 2: Кот с подсчетом мяуканий

Использован паттерн Декоратор (CounterMeow) для обертывания объекта Cat. Это позволяет отслеживать количество вызовов метода meow() без изменения класса Cat. Демонстрируется работа с методами из класса Funs.

Задание 3: Слияние упорядоченных списков

Реализован класс ListMerger с методом mergeLists(), который объединяет два упорядоченных списка в один, сохраняя упорядоченность. Проверяется, что оба списка изначально упорядочены.

Задание 4: Результаты конкурса

Программа читает данные участников из файла, вычисляет суммарные баллы, сортирует и выводит топ-3 участников, а также всех, у кого такой же балл, как у последнего из топ-3.

Задание 5: Количество уникальных букв

Класс TextAnalyzer анализирует текст и подсчитывает количество уникальных букв, игнорируя регистр. Демонстрируется использование Set для хранения уникальных значений.

Задание 6: Очередь-отражение

Класс MirroredQueue создает очередь, в которую сначала добавляются элементы из списка, а затем те же элементы в обратном порядке. Демонстрируется работа с интерфейсом Queue.

Задание 7: Построение ломаной

С использованием стримов и вспомогательного класса PolylineBuilder из набора точек формируется объект Polyline. Точки обрабатываются: Y делается положительным, убираются дубликаты, сортируются по X.

Задание 8: Группировка людей по номеру

С использованием стримов из текстового файла читаются данные, фильтруются строки без номера, имена приводятся к формату "Первая заглавная", и результат группируется по номеру.
